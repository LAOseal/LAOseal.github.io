Quick startupÂ¶

run /home/lab/libSEAL/quickStartup.py

Script to easily close the loop on SEAL static aberrations. It records the ALPAO flat commands to a calibration repository in /libSEAL.

Note

The script will ask you 4 questions:

  • Branch you are using (slm or keck)

  • If you want to use previous ALPAO flat (recommanded)

  • If the source has the right intensity

  • With which DM you want to close the loop

Warning

Check humidity on MEMS Deformable mirror before starting it (windows machine)
